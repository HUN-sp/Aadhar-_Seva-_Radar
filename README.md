## Project Overview
This project analyzes Aadhaar datasets (Enrolment, Demographic, and Biometric updates) to identify service trends. It constructs a "Master Ledger" at the pincode level to correlate service demand against supply.

**Submitted for:** UIDAI Data Hackathon 2026

## ðŸš€ How to Run (Recommended)
The project is designed to run instantly on Google Colab without manual setup.

1.  Open the file `UIDAI_Hackathon.ipynb` in this repository.
2.  Click the **"Open in Colab"** badge at the top of the notebook.
3.  In the Colab menu, select **Runtime > Run all**.

*Note: The script uses `gdown` to automatically download the required datasets from the cloud. You do not need to upload any CSV files manually.*
